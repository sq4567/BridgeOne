# Phase 1.1.3.2: 테스트 체크리스트

## ✅ 빠른 체크리스트

### 사전 준비
- [ ] ESP32-S3 펌웨어 업로드 완료
- [ ] Android 앱 빌드 및 설치 완료
- [ ] USB OTG 케이블 준비
- [ ] 시리얼 모니터 실행 (115200 baud)

### 테스트 1: 기본 연결 (5분)
- [ ] Android 앱 실행
- [ ] ESP32-S3 USB OTG 연결
- [ ] USB 권한 다이얼로그 승인
- [ ] 앱 상태: "✅ 연결됨" 표시
- [ ] ESP32 시리얼 모니터: "UART initialized" 메시지 확인

### 테스트 2: 프레임 송신 (10분)
- [ ] 터치패드 영역 한 번 탭 → 프레임 2개 전송
- [ ] ESP32 시리얼 모니터: `[RX #1] seq=0 ✓` 확인
- [ ] 오른쪽 드래그 → `deltaX > 0` 확인
- [ ] 위쪽 드래그 → `deltaY < 0` 확인
- [ ] 순번 검증 ✓ 표시

### 테스트 3: 순번 순환 (10분)
- [ ] 연속 드래그로 300+ 프레임 생성
- [ ] 순번 255 → 0 순환 확인
- [ ] ESP32 시리얼 모니터: 순환 지점 순번 검증 ✓
- [ ] 손실 프레임 0개

### 테스트 4: 1분 안정성 (15분)
- [ ] 1분간 연속 터치 입력
- [ ] 1000+ 프레임 전송
- [ ] 손실률 < 1%
- [ ] 앱 크래시 없음
- [ ] ESP32 메모리 안정 (Free Heap 변화 < 5KB)
